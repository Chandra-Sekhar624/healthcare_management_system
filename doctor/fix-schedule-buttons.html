How to Fix the Confirm and Decline Buttons in schedule.php

Note: I've created two files to help fix the Confirm and Decline buttons in the schedule.php file:
  • js/appointment-actions.js - Contains the JavaScript functions for the buttons
  • doctor/appointment-buttons.php - Contains a PHP function to generate the HTML for the buttons

Step 1: Include the JavaScript File

Add the following line to the schedule.php file just before the closing </body> tag:

<script src="../js/appointment-actions.js"></script>

Step 2: Include the PHP File

Add the following line to the schedule.php file at the top, after the other PHP includes:

// Include the appointment buttons file
include 'appointment-buttons.php';

Step 3: Replace the Appointment Actions Div

Find the appointment actions div in the schedule.php file and replace it with the following:

<?php echo generateAppointmentButtons($appointment); ?>

This should replace the entire div with class "appointment-actions" and all its contents.

Important: If you're having trouble finding the right place to make these changes, you can search for "appointment-actions" in the schedule.php file.

Step 4: Test the Buttons

After making these changes, reload the schedule.php page and test the Confirm and Decline buttons. They should now work properly!
